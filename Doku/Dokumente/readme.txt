Sofware-Umgebung:
————————————————————————————
- Toolchain herunterladen und isnatllieren
http://sourceforge.net/projects/winavr/

- Eclipse kelpler CDT runterlasen: (hat git schon integriert)
https://www.eclipse.org/downloads/packages/eclipse-ide-cc-developers/keplersr2

   - Eclipse starten. Zum Software Update Dialog gehen (Help > Install New Software ...)
   - In der Zeile Work with ... http://avr-eclipse.sourceforge.net/updatesite eintragen.
   - Button Add anklicken. Einen Namen vergeben z.B. AVR-Eclipse.
   - Neuste Version von AVR-Eclipse auswählen und installieren.

- Debugging:
http://www.mikrocontroller.net/articles/AVR_Eclipse#Debugging
http://www.mikrocontroller.net/topic/291307

- Software die für brauchbar aussieht:
http://www.sourcetreeapp.com/
